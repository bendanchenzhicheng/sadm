## Requirements

- Ruby2.0+
- Rails 4.0.2+

## Getting Start

* Clone Resosotory, `git clone git@github.com:JustQyx/sadm.git`
* run `bundle install`
* run `rake db:create` & `rake db:migrate`
* `rails s`
